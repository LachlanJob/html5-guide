HTML5

  • 什么是 HTML5

    HTML 标准的第5个修订版

    广义的 HTML5，通常指包括 HTML5、CSS3 以及其它相关的技术总和

  • 发展里程

    1990. HTML 标准建立 2000. HTML 4.01 最后一次更新 2004. WHATWG 开始专注于新的web标准 W3C 正制定 XHTML 2.0。（不向后兼容，关注可扩展性） 2008. WHATWG HTML5 第一个公开工作草案 2009. W3C 搁置 XHTML 2.0 W3C 和 WHATWG 开始一起开发 HTML5 2010. 《Thoughts on Flash》--苹果公司当时的CEO 乔布斯，发表的一篇公开文章 -- “Flash 不再是观看视频或者消费任何类型web内容，所必需的了” -- “在移动时代建立的新的开放标准，比如HTML5，将会胜利” 2012.7 W3C 专注于单一的“最终”标准 WHATWG 则认为 HTML5 应该是一个标准“快照”（永不结束、不断地更新和改善） 2014. W3C 目标：HTML5 Recommendation
