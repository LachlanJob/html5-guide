
0.  大家好，我是来自 研发中心-资讯开发组 的王沈伟，很高兴今天在这跟大家分享 html5 技术
1.1 html5 在最近的两三年内，一直是科技界媒体的最火热的话题之一。
1.2 其中移动互联网的迅速崛起，智能手机对html5的广泛支持度，在其中起了一个重要的角色。
1.3 什么是 html5，单纯的html5，就是html标准的第5个修订版。
1.4 而现在大家通常讨论的html5，是包括了html5、css3以及其它相关技术的总和。
1.5 这些技术的标准和规范，主要由2个组织在负责维护：W3C 和 WHATWG。
1.6 我们先简单回顾下 html 的发展历程。
1.6.1 html 早在1990年就已建立，它和万维网www一同诞生。
1.6.2 2000年 html 4.01 发布。
1.6.3 此后直到04年，html标准再无更新；
1.6.3.1 此时w3c正在着手制定 xhtml 2.0，注重在可扩展性，这个X上，而不打算向后兼容，此后这个决定被证明是糟糕的；
1.6.3.2 同时，由Opera、Mozilla、苹果等浏览器厂商成立了 WHATWG，开始制定新的 html 标准。
1.6.4 07年，这2个组织终于走到了一起，开始合作开发 html5。w3c 随后搁置了 xhtml 2.0 的开发。
1.6.5 08年，WHATWG 发布了第一个 html5 公开工作草案。
1.6.5.1 随后 html5 标准进展迅速，草案一边制定、公开，浏览器厂商跟着不断增加新的 html5 特性，接着开发者尝试使用它们。
1.6.6 html5 真正被大众所熟知，是在10年，苹果CEO乔布斯 发表了这篇公开文章 《Thoughts on Flash》之后。
1.6.6.1 文章阐述的最主要的2个观点就是，1. Flash 不再是观看视频或者消费任何类型web内容，所必需的了；2. 在移动时代建立的新的开放标准，比如HTML5，将会胜利。
1.6.6.2 html5 迅速占领 各大科技界媒体的头条。但是 html5 与 flash的 对比不在本次的讨论之列。
1.6.6.3 在11年末的时候，我和老党被临时抽调到德州，搞过几个月flash，ActionScript 这门语言很有意思，同时结合了 js 脚本的闭包特性，又有java的完全面向对象，以及包package引用的特点。
